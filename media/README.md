# Media Assets

This folder contains visual assets for the awesome-indie-gamedev repository.

## Current Setup

The repository currently uses the [Capsule Render API](https://github.com/kyechan99/capsule-render) for the animated header banner, which is generated dynamically and doesn't require local files.

## Optional: Custom Banner

If you'd like to use a custom banner image instead:

1. Create a banner image (recommended: 1200x400px, PNG format)
2. Name it `banner.png` and place it in this folder
3. Update the README.md to reference it: `<img src="media/banner.png" alt="Awesome Indie Gamedev" width="100%"/>`

## Banner Design Suggestions

- **Dimensions:** 1200x400px (or 3:1 aspect ratio)
- **Format:** PNG with transparency
- **Style:** 
  - Dark background (#0F172A or similar)
  - "Awesome Indie Gamedev" title
  - Game controller/gamepad icons
  - Colorful accents (#6366F1, #10B981, #F59E0B)
  - Pixel art or modern design

## Tools for Creating Banners

- [Photopea](https://www.photopea.com) - Free browser-based editor
- [Canva](https://www.canva.com) - Easy template-based design
- [Figma](https://www.figma.com) - Professional design tool
- [GIMP](https://www.gimp.org) - Free desktop editor
